# AI 学习助手后端 API 接口文档

## 项目概述

基于 FastAPI + RAG + Agent 的学习助手系统，提供作业管理和知识库问答功能。

## 基础信息

- **服务地址**: `http://localhost:8000`
- **API 前缀**: `/api`
- **接口文档**: `http://localhost:8000/docs` (Swagger UI)
- **接口文档**: `http://localhost:8000/redoc` (ReDoc)

---

### RAG 模块

| 接口路径 | HTTP 方法 | 功能描述 |
| :--- | :--- | :--- |

| 接口路径 | HTTP 方法 | 功能描述 |

## 详细接口说明

### 1. RAG 模块

#### 1.1 初始化知识库

**接口地址**: `POST /api/rag/init-knowledge-base`

**功能描述**: 重新加载知识库和向量库，用于向知识库传入新文件时调用。

**请求体**: 无

**成功响应** (200 OK):

```json
{
    "message": "知识库构建成功"
}
```

**失败响应** (500 Internal Server Error):

```json
{
    "detail": "知识库构建失败: <错误信息>"
}
```

---

#### 1.2 RAG 知识库问答

**接口地址**: `POST /api/rag/chat`

**功能描述**: 基于知识库进行非流式标准问答，接收用户提问，返回 AI 结合知识库给出的完整回答。

**请求体**:

| 字段 | 类型 | 必填 | 描述 |
| :--- | :--- | :--- | :--- |
| `question` | string | 是 | 学生的提问内容 |
| `chat_name` | string | 是 | 对话名称 |

```json
{
    "question": "什么是机器学习？",
    "chat_name": "AI基础学习"
}
```

**成功响应** (200 OK):

| 字段 | 类型 | 描述 |
| :--- | :--- | :--- |
| `answer` | string | AI 结合知识库给出的完整回答 |
| `status` | string | 状态码，成功时为 "success" |

```json
{
    "answer": "机器学习是一种人工智能技术...",
    "status": "success"
}
```

**失败响应**:

- **400 Bad Request**: 提问内容为空

```json
{
    "detail": "提问内容不能为空"
}
```

- **500 Internal Server Error**: 服务异常

```json
{
    "detail": "AI 问答服务异常: <错误信息>"
}
```

---

### 2. Agent 模块

#### 2.1 Agent 作业管理对话

**接口地址**: `POST /api/agent/chat`

**功能描述**: Agent 作业管理对话接口，支持添加作业、查询作业、更新作业状态等功能。

**请求体**:

| 字段 | 类型 | 必填 | 描述 |
| :--- | :--- | :--- | :--- |
| `question` | string | 是 | 用户的提问内容 |
| `chat_name` | string | 是 | 对话名称 |
| `chat_history` | array | 否 | 对话历史记录 |

```json
{
    "question": "帮我添加一项数学作业，截止日期是2026-06-15",
    "chat_name": "作业管理",
    "chat_history": []
}
```

**成功响应** (200 OK):

| 字段 | 类型 | 描述 |
| :--- | :--- | :--- |
| `answer` | string | Agent 给出的完整回答 |
| `status` | string | 状态码，成功时为 "success" |

```json
{
    "answer": "好的，我已经帮您添加了数学作业，截止日期为2026年6月15日。",
    "status": "success"
}
```

**失败响应**:

- **400 Bad Request**: 提问内容为空

```json
{
    "detail": "提问内容不能为空"
}
```

- **500 Internal Server Error**: 服务异常

```json
{
    "detail": "Agent 服务异常: <错误信息>"
}
```

## Agent 支持的功能

### 作业管理能力

1. **添加作业**: "帮我添加一项数学作业，截止日期是明天"
2. **查询所有作业**: "我有哪些作业？"
3. **查询未完成作业**: "我还有哪些作业没做？"
4. **更新作业状态**: "我已经完成了数学作业"
5. **学习计划**: "帮我安排一下学习计划"

### 时间处理

支持模糊时间转换，如 "明天"、"这周五" 等，Agent 会自动转换为标准的 `YYYY-MM-DD` 格式。

### 作业状态

作业状态仅有两种：
- `pending`: 未开始
- `completed`: 已完成

## 附录

### 接口调用示例

**curl 调用 RAG 问答**:

```bash
curl -X POST http://localhost:8000/api/rag/chat \
  -H "Content-Type: application/json" \
  -d '{"question": "什么是深度学习？", "chat_name": "AI学习"}'
```

**curl 调用 Agent 作业管理**:

```bash
curl -X POST http://localhost:8000/api/agent/chat \
  -H "Content-Type: application/json" \
  -d '{"question": "帮我查看所有作业", "chat_name": "作业管理"}'
```
